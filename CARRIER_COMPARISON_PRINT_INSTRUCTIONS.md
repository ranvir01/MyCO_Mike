## üìÑ Document 1: Quick Reference Poster (8.5" x 11")

### How to Print
1. Open `carrier_comparison_poster.html` in Chrome or Edge
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Select "Save as PDF" or your printer
4. Set margins to "None" or "Minimum"
5. Enable "Background graphics" checkbox
6. Print

### Paper Recommendations
| Option | Paper Type | Weight | Notes |
|--------|-----------|--------|-------|
| **Best** | Cardstock, Matte | 80-100 lb | Professional look, durable |
| **Good** | Premium Presentation Paper | 32 lb | Bright white, crisp text |
| Budget | Standard copy paper | 20 lb | Works but less durable |

### Finishing Options
- **Lamination (RECOMMENDED):** 3-5 mil laminating pouch. Protects from spills, fingerprints, and daily handling
- **Frame:** 8.5" x 11" document frame for wall mounting
- **Magnetic backing:** Attach to metal surfaces

### Placement Suggestions
- Behind the counter (visible to staff)
- On the wall near checkout area
- Inside a binder for reference

## üìë Document 2: Customer Handout (5.5" x 8.5")

### How to Print
1. Open `carrier_comparison_handout.html` in Chrome or Edge
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. **Page Setup:**
   - Paper size: Letter (8.5" x 11")
   - Orientation: Landscape
   - Scale: 100%
4. Enable "Background graphics" checkbox
5. Print

### Paper Recommendations
| Option | Paper Type | Weight | Notes |
|--------|-----------|--------|-------|
| **Best** | Cardstock, Matte | 65-80 lb | Professional, memorable |
| **Good** | Brochure Paper | 32 lb | Semi-gloss, good feel |
| Budget | Premium copy paper | 24 lb | Adequate for handouts |

### Cutting Instructions
1. Print on 8.5" x 11" paper (landscape orientation)
2. Cut down the center vertically (at 5.5")
3. Result: Two 5.5" x 8.5" handouts per sheet

### Tip for Bulk Printing
- Print 25-50 sheets at a time
- Use a paper cutter for clean, straight cuts
- Keep a stack ready at the counter

### Adding a QR Code
The handout has a placeholder for a QR code. To add your actual QR code:
1. Generate a QR code linking to your trade-in calculator page
2. Use [QR Code Generator](https://www.qrcode-generator.de/) or similar
3. Replace the placeholder text with the actual QR image
4. Size: approximately 0.8" x 0.8"

## üìö Document 3: Laminated Flip Chart

### How to Create

#### Option A: Print from Markdown
1. Open `CARRIER_COMPARISON_FLIP_CHART.md`
2. Copy each page section
3. Format in Word/Google Docs with large fonts
4. Print each page on cardstock

#### Option B: Use the HTML/CSS Tables
1. Create a new HTML document for each flip chart page
2. Use the table styling from the poster file
3. Print each page at 5" x 7" or 6" x 8"

### Paper & Lamination
| Component | Specification |
|-----------|--------------|
| Paper | 100 lb cardstock (bright white) |
| Lamination | 5-7 mil pouches (thicker = more durable) |
| Size | 5" x 7" or 6" x 8" (index card to half-letter) |

### Binding Options
| Method | Pros | Cons |
|--------|------|------|
| **Ring binder (RECOMMENDED)** | Easy to flip, lays flat | Takes up counter space |
| Spiral binding | Compact, professional | Requires binding machine |
| Metal rings (corner) | Simple, inexpensive | May tear over time |
| Easel stand | Visible to customers | Most expensive option |

### Assembly
1. Print all 5 pages on cardstock
2. Laminate each page
3. Punch holes (top or left side)
4. Bind with rings or spiral
5. Optional: Add a cardstock cover with "Trade-In Cheat Sheet" title
